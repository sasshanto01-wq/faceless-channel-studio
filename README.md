[![GitHub Pages](https://github.com/sasshanto01-wq/faceless-channel-studio/actions/workflows/deploy.yml/badge.svg)](https://github.com/sasshanto01-wq/faceless-channel-studio/actions/workflows/deploy.yml)

An interactive web tool for building viral faceless YouTube channels. Complete 9-step system from niche selection to monetization, powered by Claude AI.

## 🚀 Features

- **9-Step Roadmap** — From niche selection to revenue maximization
- **AI-Powered Prompts** — 4 Claude AI integrations for strategy, visuals, SEO, and monetization
- **Complete Tool Stack** — Pre-curated tools and integrations
- **Interactive UI** — Beautiful dark-themed interface with smooth navigation
- **Video Export** — Download your entire strategy as a text file

## 📋 Steps Included

1. **Best Faceless Niches** — Learn the most profitable niches (sleep videos, LoFi beats, rain sounds)
2. **Planning Prompt** (AI) — Generate channel strategy and audience analysis
3. **Image Generation** — Tools and guides for AI-generated backgrounds
4. **Visual Prompts** (AI) — Generate 5 production-ready Midjourney prompts
5. **Video Assembly** — Learn to combine AI images with ambient audio
6. **SEO Package** (AI) — Complete YouTube SEO optimization
7. **Monetization Ways** — AdSense, affiliate links, and more
8. **Revenue Maximizer** (AI) — Strategy to maximize RPM and earnings
9. **Tool Stack** — Complete list of free and paid tools
